Place a Live2D Cubism 3/4 model here for Haruhi-style desk pet.

Expected entry file (example):
  haruhi.model3.json

You must also satisfy Live2D Cubism SDK for Web licensing and include the Cubism Core runtime
as required by pixi-live2d-display (see project README).

If this folder has no model, the app shows a Pixi-based emoji stage instead.
